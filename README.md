# 🌍 Mapa do Intercambista

O **Mapa do Intercambista** é uma plataforma desenvolvida para **conectar intercambistas e agências de intercâmbio em um único lugar**.
O objetivo do projeto é facilitar a busca por programas de intercâmbio, permitir que agências divulguem seus pacotes e criar uma comunidade onde usuários possam compartilhar experiências.

Este projeto está sendo desenvolvido como **aplicativo mobile Android** utilizando **Java e Android Studio**.

---

# 🚀 Funcionalidades

Funcionalidades planejadas:

* 🌎 Catálogo de pacotes de intercâmbio
* 🏫 Perfis de agências de intercâmbio
* 💬 Fórum estilo rede social para compartilhar experiências
* ⭐ Sistema de avaliações de agências e países
* 📚 Tutoriais sobre vistos e documentação
* 🎓 Recomendações de cursos e preparação cultural

---

---

# 🎯 Objetivo do Projeto

O objetivo do **Mapa do Intercambista** é se tornar uma plataforma central para:

* Encontrar oportunidades de intercâmbio
* Comparar agências
* Compartilhar experiências reais de viagem
* Ajudar novos intercambistas a se prepararem melhor
